# only1word
#### only one word story

一个简易的评论型网站，思路是网易评论，也就是盖楼，有些神评论一层一层下来可以组成一个故事了，想法也就来源于此

一人一句话，一个小故事慢慢就出来了

* CSS框架: sematic-ui-2.2
* python: 3.5.2
* Django: 1.11
* MySQL: 5.6
* jquery: 3.2.1

现在主要目的就两个：发帖（写故事）,跟帖（评论）

目前的功能：
1. 注册用户
2. 登录
3. 发帖
4. 评论
5. 收藏
6. 取消收藏
7. 按标题，标签，作者搜索
8. 个人中心：
  1. 统计范围：我的文章，我的收藏，我的评论
  2. 修改密码


计划实现：
1. 生成整篇文章
2. 用户信息修改：用户名，头像，邮箱
3. 注册邮箱验证
